# Glossário de Termos

## A

**API (Application Programming Interface)**
Interface que permite comunicação entre diferentes sistemas de software.

**ADR (Architecture Decision Record)**
Documento que registra uma decisão arquitetural importante e seu contexto.

## B

**Badge**
Conquista desbloqueável no sistema de gamificação ao cumprir critérios específicos.

**Build**
Processo de compilação e empacotamento do código para produção.

## C

**CDN (Content Delivery Network)**
Rede de servidores distribuídos para entrega rápida de conteúdo.

**Check-in**
Registro de presença do usuário em um ponto de interesse.

**CI/CD (Continuous Integration/Continuous Deployment)**
Práticas de automação de integração e deploy de código.

**Context API**
Sistema nativo do React para gerenciamento de estado global.

## D

**Deploy**
Processo de publicação da aplicação em um ambiente de produção.

**DPO (Data Protection Officer)**
Encarregado de Proteção de Dados conforme LGPD.

## G

**Gamificação**
Aplicação de elementos de jogos em contextos não-jogos para engajamento.

**GPS (Global Positioning System)**
Sistema de posicionamento global usado para localização.

## H

**HMAC (Hash-based Message Authentication Code)**
Código de autenticação de mensagem baseado em hash para verificação de integridade.

**Hook**
Função especial do React que permite usar estado e outros recursos.

## J

**JWT (JSON Web Token)**
Padrão para tokens de autenticação em formato JSON.

## L

**LGPD (Lei Geral de Proteção de Dados)**
Lei brasileira de proteção de dados pessoais (Lei nº 13.709/2018).

**Leaderboard**
Ranking de usuários ordenado por pontuação.

## M

**Mock**
Simulação de dados ou serviços para desenvolvimento e testes.

**MVP (Minimum Viable Product)**
Versão mínima de um produto com funcionalidades essenciais.

## P

**POI (Point of Interest)**
Ponto de interesse turístico no sistema.

## Q

**QR Code (Quick Response Code)**
Código de barras bidimensional que armazena informações.

## R

**RBAC (Role-Based Access Control)**
Controle de acesso baseado em papéis do usuário.

**Route (Rota)**
Roteiro turístico composto por múltiplos POIs.

## S

**SPA (Single Page Application)**
Aplicação web que carrega uma única página e atualiza dinamicamente.

**Session Storage**
Armazenamento temporário do navegador que persiste durante a sessão.

## T

**TypeScript**
Superset de JavaScript com tipagem estática.

## V

**Vercel**
Plataforma de deploy e hosting para aplicações frontend.

**Vite**
Ferramenta de build rápida para projetos JavaScript/TypeScript.
